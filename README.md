# File Sharing Platform with Encryption

Учебный проект платформы для обмена файлами с поддержкой шифрования.

## Возможности

- создание пользователя
- загрузка файлов
- шифрование файлов
- автоматическое тестирование

## Тестирование

Для запуска тестов:

python -m unittest discover file-sharing-platform/tests
